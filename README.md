# NNN-CVE

NNN-CVE is an automated web vulnerability scanning tool developed in Python.  
It combines multiple security scanners to identify vulnerabilities in web applications and generates structured security reports.

---

## Features

- Automated web vulnerability scanning
- Integration with multiple security tools
- Generates vulnerability assessment reports
- Modern and user-friendly UI
- Modular and extensible Python codebase
- Suitable for learning and authorized security testing

---

## Tools Used

- Python
- Nmap
- Nikto
- Nessus

---

## Installation

1. Clone the repository

2. Install required Python packages
 - pip install -r requirements.txt
    
3. Ensure the following tools are installed and configured
- Nmap
- Nikto
- Nessus

---

## Usage

Run the scanner with a target URL or IP address

---

## Output

- Generates security reports in structured formats such as HTML or JSON
- Reports include discovered vulnerabilities and scanner-specific findings
